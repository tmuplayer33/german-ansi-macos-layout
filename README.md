# Simple German Keyboard Layout for ANSI Keyboard
Umlaute können mit dem option key + der entsprechenden Taste geschrieben werden. 
Beispiel:

```
option + a = ä
option + o = ö
option + s = ß
```
Wenn zusätzlich Shift gedrückt wird, werden die Umlaute groß geschrieben:
```
option + shift + a = Ä
```

Weitere mappings:
```
option + 4 = €
```
